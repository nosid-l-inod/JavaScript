Unordered Lists

In this list, the order of the items does not matter

  • Milk
  • Sugar
  • Bread

Ordered Lists

In this list the order of the element matters

  1. Drive to the end of the road
  2. Turn right
  3. Go straight to the first whatever
  4. Turn left
  5. The school is in 150 meters

Nesting Lists

We can nest lists inside another one

  1. Remove all the skin from the garlic.
  2. Remove all the seeds
  3. Process all the ingredients
    • If you want a coarse something
    • If you want it smooth do it another way
